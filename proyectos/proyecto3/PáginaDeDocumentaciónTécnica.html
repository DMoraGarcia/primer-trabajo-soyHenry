introduction

Python is a high-level programming language created by Guido van Rossum in the late 1980s and early 1990s. It stands out for its simplicity and readability, making it ideal for beginners and professionals alike. Python Key Features:

  • Readability: Python focuses on code readability, meaning it uses a writing style that makes programs easy to understand and maintain. Its syntax uses English keywords that are intuitive and simple.
  • Versatility: It is a versatile language that can be used in various fields, from web development and desktop applications to data analysis, artificial intelligence, machine learning and much more.
  • Great community and libraries:Python has an active community that has developed a wide range of libraries and modules to perform various tasks, making it easy to find tools and solutions for specific projects.
  • Interpreted and interactive: Python is an interpreted language, meaning it does not require a compilation step before execution. This enables faster development by allowing instructions to be executed individually in an interactive environment known as the Python interpreter.
  • Portability: Python is compatible with most operating systems, meaning that the same code can run on different platforms without significant modifications.

Python and C++

Python and C++ are two programming languages ​with quite different features and approaches. Here I present a detailed comparison between Python and C++ in several aspects:

  1. Syntax and Readability:
  • Python: It is known for its simple and readable syntax. It uses clear syntax with extensive use of indentation to define blocks of code. Its readability makes learning and understanding the code easier.
  • C++:It has a more complex and detailed syntax compared to Python. It requires the use of braces "{}" to delimit blocks of code and uses a static typing system that may be less intuitive for beginners.
  1. Typing:
  • Python:It is a dynamically typed language, which means that it is not necessary to declare the data type of a variable before using it. Variables can change type during program execution.
  • C++:It is a statically typed language, which implies that the data type of the variables must be declared before use. The type of a variable does not change once it has been defined.
  1. Speed ​​and Performance:
  • Python:It is known to be slower compared to low-level languages ​​like C++. Python is interpreted, meaning that instructions are executed line by line. However, some Python libraries may be written in C or C++ to improve performance.
  • C++:It is a low-level language and offers greater control over memory management and performance. Its compilation produces highly optimized machine code, which generally results in faster execution.
  1. Easy to use:
  • Python:It is easier to learn and use due to its simple syntax and readability. It is ideal for beginners and rapid prototyping.
  • C++:Has a steeper learning curve due to its complexity and more detailed handling of memory and pointers. It may be more challenging for beginners, but offers greater control over hardware and optimization.
  1. Applications and Domains:
  • Python:It is widely used in web development (with frameworks such as Django or Flask), data science, artificial intelligence, machine learning, scripting, automation, among others.
  • C++:It is commonly used in the development of operating systems, video games, high-performance software, embedded systems and applications that require precise control over system resources.
  1. Memory management:
  • Python:Automatically handles memory management, which makes it easier to write code but can lead to less efficient use of resources.
  • C++Requires manual memory management with concepts such as memory allocation and freeing, which provides greater control but can also lead to errors such as memory leaks or invalid memory references if not handled correctly.

In summary, Python stands out for its simplicity, readability and ease of use, ideal for rapid development and high-level applications, while C++ offers greater control, performance and is well suited for low-level applications where use is needed. resource efficient. The choice between both languages ​​depends on the needs of the project, the level of control required and the preferences of the programmer.

Variables

Variables in Python are containers of information! They are tags that are used to store data in the computer's memory. Here are details about variables in Python:

  1. Declaration and Assignment:

    2. In Python, variables are created by assigning a value to them using the "=" assignment operator. You do not need to declare the variable type explicitly. For example:

x = 5
nombre = "Ejemplo"
  1. Variable Names:
    • They must begin with a letter or an underscore ("_").
    • They can contain letters, digits, and underscores.
    • Python is case sensitive in variable names ("my_variable" is different from "my_Variable").
  1. Dynamic Data Types:

    2. Python is dynamically typed, meaning that the data type of a variable is automatically inferred based on the value it contains. For example:

x = 5
x = "Hola"
  1. Variable Reassignment:

    2. Variables can change type or value during program execution.

y = 10
y = y + 5
  1. Keywords:

    2. Some words are reserved and cannot be used as variable names, as they have a special meaning in Python. For example: "if", "else", "for", "while", "True", "False", "None", among others.

  1. Scope of Variables:

    2. The scope of a variable refers to the parts of the code where the variable is accessible. Python has local and global scope rules. Variables defined inside a function have local scope, while those defined outside all functions have global scope.

  1. Deleting Variables:

    2. The del function is used to remove a reference to a variable and free the memory it occupied.

z = 7
del z
  1. Naming Conventions:

    2. A naming convention called snake_case (lower case separated by underscores) is followed to name variables in Python. For example: "my_variable", "full_name".

    Variables in Python are essential for storing and manipulating data. Their flexibility and dynamism make them powerful for the development of programs in this language.

Variable scope

The scope of a variable in Python refers to the region of the code where that variable is accessible. Python has clear variable scope rules that determine where a variable can be used or referenced.

Scope Types in Python:

  1. Local Reach:
    • Local variables: These are variables defined within a function and are only accessible within that function. They cannot be used outside of that function. Example:
def mi_funcion():
x = 10
print(x)
mi_funcion()
  1. Global Reach:
    • Global variables: These are those defined outside of any function or in a global context. They can be accessed and modified both inside and outside of functions. Example:
y = 20
def otra_funcion():
print(y)
otra_funcion()
print(y)
  1. Scope of Local and Global Variables:
    • If a variable is defined both locally and globally with the same name, the local scope takes precedence within the function. Example:
z = 30
def funcion():
z = 40
print(z)
funcion()
print(z)
  1. "Global" Keyword:
    • The "global" keyword is used to indicate that a variable inside a function is the same global variable defined outside the function. Example:
w = 50
def modificar():
global w
w = 60
print(w)
modificar()
print(w)

These scope rules in Python are essential for understanding how variables behave inside and outside functions. Clarity in the scope of variables makes it easier to understand and maintain the code.

Global variables

Global variables in Python are those defined outside of any function or in a global scope. These variables can be accessed and modified both inside and outside of functions in the program. Here are details about global variables in Python:

Definition of Global Variables:

variable_global = 10

In this example, "global_variable" is defined outside of any function, making it a global variable. It can be accessed and modified from any part of the program, both inside and outside of functions.

Access to Global Variables from a Function:

Global variables can be read into a function without declaring them global. However, if you are attempting to modify a global variable within a function, you must explicitly declare that variable as global using the global keyword.


Reading Global Variables:

variable_global = 10
def mi_funcion():
print(variable_global)
mi_funcion()

Modifying Global Variables using the global keyword:

variable_global = 10
def modificar_variable():
global variable_global
variable_global = 20
print(variable_global)
modificar_variable()
print(variable_global)

Considerations on the use of Global Variables:

  1. Avoid excessive use: Excessive use of global variables can make the code difficult to understand and lead to errors that are difficult to trace.
  2. Impact on readability and maintainability: Global variables can be modified from anywhere in the program, which can make tracking changes in value more complex.
  3. Modify with caution: If global variables are modified from functions, it is essential to keep a clear track of the changes to avoid unwanted side effects on the code.

Global variables can be useful, but it is recommended that they be used with caution to keep the code clear and maintainable. In many cases, local variables passed as parameters to functions are preferred rather than relying too much on global variables.

Constants

In Python, there is no specific data type for constants like in some other programming languages. However, a convention is followed to declare variables whose value is considered constant during program execution.

Convention for Constants in Python:

Constants in Python are named in uppercase letters to indicate that their value should not be modified throughout the program. Although these variables can be reassigned, programmers are expected not to do so due to the naming convention.
Example:

PI = 3.14159
LIMITE_SUPERIOR = 100
NOMBRE_EMPRESA = "Mi Empresa"

Good Practices for Using Constants:

  1. Clear naming: Constants are named in capital letters and with underscores to improve the readability of the code and to emphasize that they should not be modified.
  2. Avoid reassignment: Although Python allows variables to be reassigned, it is recommended not to modify variables that have been defined as constants to maintain the integrity of the code.
  3. Clear documentation: It is helpful to add comments or documentation that explains the purpose and use of constants in the code.

Practical Approach:

Although Python does not have a specific data type for constants, using the uppercase naming convention helps developers identify and treat these variables as constants. Constants are useful for defining values ​​that do not change during program execution and provide clear context about their meaning and use in code.

Remember that these conventions are recommended practices, but they do not prevent variables defined as constants from being reassigned. The main emphasis is on code clarity and communicating the intention that certain values ​​should not be modified.

Data types

In Python, data types define the characteristics of the values ​​that can be stored and manipulated in code. Python is a dynamically typed language, meaning that the data type of a variable is automatically inferred based on the value assigned to it. Here is a detailed description of data types in Python:

Basic Data Types:

  1. Numbers:
    • int: Integers without decimal point (positive or negative).
    • float: Decimal numbers (positive or negative).
    • complex:Complex numbers with a real and imaginary part (x + yj).
  1. Booleans:
    • bool: Represents a truth value: True or False.
  1. Character Strings:
    • str: Character sequences enclosed in single or double quotes. Examples: "Hello", 'Python'.

Composite Data Types:

  1. Lists:
    • list: Ordered and modifiable collections of elements. They can contain different types of data. They are defined between square brackets [].
  1. Tuples:
    • tuple: Ordered and immutable sequences of elements. They are defined in parentheses ().
  1. Dictionaries:
    • dict: Unordered collections of key-value pairs. They are defined in curly braces {} and have the key:value structure.
  1. Sets:
    • set: Unordered collections without duplicate elements. They are defined between braces {}

Other Types of Data:

  1. NoneType:
    • None: Data type that represents the absence of a value. It is used to indicate the lack of return from a function or the lack of value assigned to a variable.

Key Features:

  • Dynamic: Variables do not require an explicit type declaration. The type is automatically inferred.
  • Mutability: Some data types are mutable (such as lists and dictionaries) and can be modified after creation, while others are immutable (such as tuples and strings) and cannot be modified.
  • Sequence and Collections: Python provides data types to handle ordered sequences and unordered collections of data.

Conversion between Data Types:

Python offers built-in functions to convert between different data types, such as int(), float(), str(), list(), tuple(), dict(), set(), among others.

Practical Use:

Knowledge of data types is essential for manipulating data and structuring code in Python. Understanding these types and their properties helps you write more efficient and effective programs.

if...else statement

The if...else statement in Python is a control flow structure that allows decisions to be made based on Boolean conditions. This structure allows one block of code to be executed if a condition is true (True) and another block of code if that condition is false (False). Here is a detailed explanation of if...else:

if...else syntax:

if condicion:
else:

Example:

edad = 18
if edad >= 18:
print("Eres mayor de edad")
else:
print("Eres menor de edad")

Explanation:

  • The condition is evaluated after the if. If the condition is True, the first block of code is executed.
  • If the condition is false, the if block is skipped and the code block after the else is executed.

if...elif...else (with multiple conditions):

Multiple if blocks can be used using the elif keyword (short for "else if") to evaluate multiple conditions.

numero = 7
if numero > 0:
print("El número es positivo")
elif numero == 0:
print("El número es cero")
else:
print("El número es negativo")

Nesting of if...else:

It is possible to nest if...else structures within others to evaluate more complex conditions.

x = 10
if x > 5:
if x < 15:
print("x está entre 5 y 15")
else:
print("x es mayor que 15")
else:
print("x es menor o igual que 5")

Key points:

  • The if...else is a fundamental tool to control program flow based on Boolean conditions.
  • Multiple if and else can be nested to handle more complex situations.
  • It is important to note indentation, since Python uses indentation to determine the blocks of code associated with each control flow structure.

The if...else is essential for making logical decisions in Python programs, allowing code to be executed conditionally based on evaluated conditions.

while statement

The while statement in Python is a control flow structure used to repeat a block of code as long as a specific condition is true. This structure is called a loop because it repeatedly executes a set of statements as long as the condition evaluates to True. Here is a detailed explanation of while:

Sintaxis del while:

while condicion:

Example:

contador = 0
while contador < 5:
print(f"El contador es: {contador}")
contador += 1

Explanation:

  • The condition is evaluated after the while. If the condition is True, the indented code block is executed.
  • After the code block is executed, the condition is evaluated again. If the condition is still true, the cycle repeats. This process continues until the condition evaluates to False.
  • It is important to ensure that the condition in the while eventually returns False to avoid an infinite loop.

While Loop Control:

  • break: Used to exit the while loop prematurely if a specific condition is met.
contador = 0
while True:
print(f"El contador es: {contador}")
contador += 1
if contador >= 5:
break
  • continue: Se usa para pasar a la siguiente iteración del bucle while sin ejecutar el resto del código en el bloque actual, si se cumple una condición específica.
contador = 0
while contador < 5:
contador += 1
if contador == 3:
continue
print(f"El contador es: {contador}")

Key points:

  • The while loop is useful when the number of iterations is not known in advance and depends on a condition.
  • You should use caution when using a while loop, because if the condition never evaluates to False, it can cause an infinite loop and block program execution.

The while loop is a powerful tool for repeating blocks of code in Python as long as a specific condition is met, providing flexibility in executing repetitive tasks.

Function declarations

In Python, functions are defined using the def keyword. Functions are reusable blocks of code that perform a specific task when called. Here is the syntax and details on how to declare functions in Python:

Function Declaration Syntax:

def nombre_de_la_funcion(parametros):
"""
Docstring: Descripción opcional de la función.
"""

return resultado
  • def: It is the keyword used to declare a function.
  • function_name: It is the name you give to the function. It should follow Python naming rules and be descriptive of the purpose of the function.
  • parameters: They are variables used to receive values ​​that are passed to the function. They can be optional.
  • Docstring: It is a comment that describes the purpose and operation of the function. It's optional, but it's good practice to include it to document the feature.
  • Function Body: Contains one or more instructions that are executed when the function is called.
  • return: Optional keyword that indicates the value that the function will return as a result. A function can have multiple return statements, but only one of them will be executed.

Simple Function Example:

def suma(a, b):
"""
Esta función devuelve la suma de dos números.
"""
resultado = a + b
return resultado

Calling a Function:

After defining a function, you can call it from anywhere in the program to execute the code within that function.

resultado_suma = suma(3, 5)
print(resultado_suma)

Function Parameters:

Functions can have parameters which are variables used to receive values ​​that are passed to the function when it is called. These parameters are optional and can have default values.

def saludo(nombre="Usuario"):
print(f"Hola, {nombre}!")
saludo()
saludo("Juan")

Value Returned by the Function:

Functions can return values ​​using the return keyword.

def multiplicacion(a, b):
return a * b
resultado_mult = multiplicacion(4, 5)
print(resultado_mult)

Key Features:

  • Functions allow the code to be modularized, making it more readable, maintainable and reusable.
  • Parameters allow functions to accept different data to perform various tasks.
  • The return allows functions to return values ​​that can be used in other parts of the program.

Functions in Python are a fundamental part of programming and provide an organized way to execute repetitive or specific code. It is important to understand how to declare, call, and use functions to write effective Python programs.
